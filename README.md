### 💡 프로젝트 다운로드 & 실행하는 방법

#### 1. 레포지토리 클론하기

```
git clone https://github.com/Seokyung/Seokyung-Portfolio.git
```

#### 2. 프로젝트 파일과 패키지 다운로드하기

```
npm install
```

#### 3. 프로젝트 시작하기!

```
npm run dev
```
